# Meu Todo Digital Simples - Página de Vendas

Esta é a página de vendas para o aplicativo **Meu Todo Digital Simples**, uma ferramenta simples e intuitiva para gerenciar tarefas diárias.

## 📋 Conteúdo

- **index.html** - Página principal de vendas
- **styles.css** - Estilos responsivos
- **README.md** - Este arquivo

## 🚀 Como colocar no ar

### Opção 1: GitHub Pages
1. Vá até as configurações do repositório
2. Em "Pages", selecione "main" como branch
3. A página estará disponível em: `https://jprodrigues9988-collab.github.io/M-todo-Digital-Simples/`

### Opção 2: Netlify
1. Acesse [netlify.com](https://netlify.com)
2. Faça login ou crie uma conta
3. Clique em "Add new site" → "Import an existing project"
4. Selecione este repositório GitHub
5. Configure e deploy

### Opção 3: Vercel
1. Acesse [vercel.com](https://vercel.com)
2. Faça login ou crie uma conta
3. Clique em "New Project"
4. Selecione este repositório GitHub
5. Configure e deploy

## 🎨 Personalização

Você pode personalizar os seguintes elementos:

- **Cores**: Edite as variáveis CSS em `styles.css` (`:root { --primary-color: ... }`)
- **Textos**: Modifique o conteúdo em `index.html`
- **Preços**: Atualize os planos na seção de pricing

## 📱 Responsivo

A página é totalmente responsiva e funciona bem em:
- Desktop (1200px+)
- Tablet (768px - 1199px)
- Mobile (< 768px)
